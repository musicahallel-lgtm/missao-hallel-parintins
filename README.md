# Missão Hallel Parintins

Mini site da Missão Hallel Parintins — Comunidade Católica Hallel.

> "Ser resgate de almas para Deus, através do sacrifício de louvor."

## Arquivos

- `index.html` — o site completo (página única).
- `logo.png` — logo da Comunidade Hallel.
- `foto1.jpg` … `foto4.jpg` — fotos da galeria.

## Como atualizar as imagens

Basta substituir os arquivos `logo.png` e `foto1.jpg`–`foto4.jpg` por novas
imagens **com os mesmos nomes** e enviar para o GitHub.
